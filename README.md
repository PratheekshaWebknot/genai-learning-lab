# GenAI Learning Lab

This repository contains hands-on assignments and projects built while learning
Generative AI concepts and tools, including LLMs, LangChain, LangGraph, FastAPI,
RAG systems, and production patterns.

## Structure
Each folder corresponds to a specific learning day or module.

## Tech Stack
- Python
- HuggingFace Transformers
- API-based LLMs
- FastAPI
- LangChain & LangGraph
- Docker

## Status
🚧 Work in progress — updated incrementally per learning module.
